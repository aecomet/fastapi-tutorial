# アーキテクチャ

## 技術スタック

| 役割 | ツール | バージョン |
|------|--------|-----------|
| Web フレームワーク | FastAPI | >=0.135.2 |
| ASGI サーバー | Uvicorn | >=0.42.0 |
| データバリデーション | Pydantic | (FastAPI 依存) |
| パッケージ管理 | uv | - |
| 言語 | Python | >=3.14 |

## コンポーネント構成

```
クライアント (ブラウザ / curl)
        │
        │ HTTP
        ▼
┌───────────────────┐
│    Uvicorn        │  ASGI サーバー
│  (ポート 8000)    │  リクエストの受付・プロセス管理
└────────┬──────────┘
         │
         ▼
┌───────────────────┐
│    FastAPI        │  Web フレームワーク
│    (Starlette)    │  ルーティング・バリデーション・シリアライズ
└────────┬──────────┘
         │
         ▼
┌───────────────────┐
│    main.py        │  アプリケーションロジック
│  エンドポイント    │  ビジネスロジック・レスポンス生成
└───────────────────┘
```

## ファイル構成

```
fastapi-tutorial/
├── docs/
│   └── architecture.md  # このファイル
├── main.py              # FastAPI アプリ本体・エンドポイント定義
├── pyproject.toml       # プロジェクトメタデータ・依存関係
├── uv.lock              # 再現性確保のためのロックファイル
└── README.md            # セットアップ・起動手順
```

## リクエストフロー

```
GET /  リクエスト
    │
    ▼
Uvicorn がリクエストを受信
    │
    ▼
FastAPI がルートマッチング ( GET "/" )
    │
    ▼
read_root() 関数を呼び出し
    │
    ▼
{"message": "Hello World"} を JSON で返却
```

## エンドポイント一覧

| メソッド | パス | 説明 |
|---------|------|------|
| GET | `/` | Hello World レスポンスを返す |
| GET | `/docs` | Swagger UI（自動生成） |
| GET | `/redoc` | ReDoc（自動生成） |
| GET | `/openapi.json` | OpenAPI スキーマ（自動生成） |

## 開発サーバーとプロダクション

| 項目 | 開発 | プロダクション（例） |
|------|------|-----------------|
| 起動コマンド | `uv run uvicorn main:app --reload` | `uvicorn main:app --host 0.0.0.0 --workers 4` |
| ホットリロード | あり | なし |
| ワーカー数 | 1 | CPU コア数に応じて設定 |
